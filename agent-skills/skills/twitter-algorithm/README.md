# Twitter Algorithm

Optimize tweets for organic reach using insights from Twitter's open-source algorithm.

## What It Does

Provides evidence-based strategies for maximizing tweet visibility based on analysis of Twitter's source code. Includes golden rules, pre-post checklist, content guidelines, and timing strategies for better organic reach.

## Quick Start

```bash
# Score a draft tweet
./scripts/score-tweet.sh "Your tweet text here"

# Analyze posted tweet performance
./scripts/analyze-tweet.sh <tweet_id>

# Find optimal posting times
./scripts/best-time.sh
```

## Key Rules

- **8-hour half-life** — Early engagement compounds
- **Replies > Quotes > RTs > Likes** — Prioritize high-signal engagement  
- **Native media wins** — Upload images/video directly
- **0-1 hashtags** — More triggers spam detection

## Requirements

- Node.js 18+
- Access to Twitter's algorithm source code references
